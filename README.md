Opis aplikacji:
Aplikacja volleybase jest bazą siatkarzy.

Aplikacja oferuje:

- tworzenie użytkownika
- logowanie na konto utworzonego użytkownika
- dodawanie zawodników
- wyświetlanie dodanych zawodników

Stack technologiczny:

FRONTEND

- React
- React Router DOM
- Formik
- Styled Components
- Axios
- Ant
- Storybook
- Jest
- React testing library

BACKEND

- Firebase
